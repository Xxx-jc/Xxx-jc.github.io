我的blog搭建过程

发表于 2019-10-09 更新于 2019-10-12

#运用Hexo+Github构建
#通过Hexo生成静态网页再将其托管于Github上
$何为Hexo？

##Hexo是一个快速、简洁高效的博客框架。Hexo使用markdown及其他渲染引擎解析文章

$何为Github

##Github是一个面向开源及私有软件项目的托管平台，因为只支持git作为唯一的版本格式进行托管，故名Github

1.
$安装 Node.js
https://nodejs.org/en/

#安装之后通过cmd（win）确认

###如果已经安装该软件，则会显示对应的安装软件版本，否则将会提示“xxx”不是内部命令或外部命令（下面安装的软件同理）

2.
$安装Git
https://git-scm.com/downloads

###安装成功后点击鼠标右键就会看到两个新增的选项：Git Gui Here和Git Bash Here

3.
$创建Hexo文件夹
在你觉得合适的磁盘里新建一个文件夹作为blog目录(例：E:\Hexo)直接在文件夹内点击鼠标右键，选择Git Bash

3.1
$安装Hexo
https://hexo.io/
利用npm命令进行安装
在任意位置点击鼠标右键，选择Git Bash Here

###可以通过命令查询版本（在cmd下似乎需要设置环境变量不然会报错，直接用gitbash即可）
###后面的步骤都是要在创建的blog目录下启动GitBash

3.2
$查看效果
截至目前我们已经搭建起本地的blog了，接着在你的blog目录下启动GitBash

复制到浏览器即可

#至此，本地blog已经搭建起来了，但此时你的blog只能在本地访问，接下来我们要做的就是将blog部署到Github

4.
$将博客部署到Github
https://github.com/

4.1
$注册Github账号

注册成功后会出现“Create a new repository”页面
往下看 owner是你github的用户名
repository name是你的仓库名称
#这里的仓库名称必须按xxx.github.io填写（xxx是你自己的用户名）

4.2
$创建ssh
打开GitBash

接着一路回车 最后退出即可

接下来打开c盘——>用户——>.ssh——>id_rsa.pub(用记事本方式打开)
复制粘贴里面的ssd到你的Github中
#登录Github，点击右上角的个人头像，点击settings，接下来在Personal settings里点击SSH and GPG keys
#Title就写你的用户名就好，下面的key需要把你复制的SSH粘贴上去，完事后添加就好

4.3
$修改站点配置文件（_config.yml)
可以使用sublime等文本编辑器更改

首先修改你的url（也就是你github的地址）

接着按图片修改（地址改成你Github仓库的地址）

#在正式部署之前要先进行模块安装
还是老样子在你的blog目录下打开Git Bash并输入命令
#安装模块

#修改用户名和邮箱

#最后执行下列指令完成部署

输入完成后会弹出一个窗口，输入你的Github账号即可

6.
到这里最基本的步骤已经完成了，输入你github.io的地址即可访问
